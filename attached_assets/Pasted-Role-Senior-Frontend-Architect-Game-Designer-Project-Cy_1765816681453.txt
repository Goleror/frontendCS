### 2. CORE GAME ENGINE (Logic)
Create a `useGameEngine` hook managing:
1.  **FileSystem:** Nested JSON objects. Files have `content` and `permissions` (e.g., "r" or "rw").
2.  **Processes:** Array `{pid, name, cpu, user, protected}`. (Added 'protected' flag).
3.  **Command Parser:**
    - `ls`, `cd`, `pwd`, `clear`, `help`
    - `cat [file]`, `rm [file]`, `touch [file]`, `echo [text] > [file]`
    - `ps`, `kill [pid]`
    - `grep [string] [file]` (Search text in file)
    - `chmod +w [file]` (Unlock file for deletion - CRITICAL)
    - `submit_report [text]`

**Logic Rules for Hardcore Mode:**
- If user tries `kill [pid]` on a `protected: true` process, return Error: "PROTECTED_PROCESS: Daemon active. Deactivation key required in /etc/killswitch".
- If user tries `rm [file]` where permissions are "r" (read-only), return Error: "PERMISSION DENIED. File is read-only."
- User must run `chmod +w [file]` to change permissions to "rw" before deleting.
